# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-15

### Added
- **Interactive UI Dashboard**: Sleek futuristic twilight theme styled with Tailwind CSS v4, supporting collapsible view options and full-screen bento layout grids.
- **Cognitive AI Generation**: Integrated robust support for `@google/genai` on server-side proxies, preventing key leaks to client threads.
- **SQL History persistence**: Implemented transactional history storage on SQLite using standard `sqlite3` libraries.
- **Conventional Commits**: Built prompt classifiers to identify change scopes (Feature, Fix, Docs, Refactor etc.) and format proper Git headers.
- **Pull Request Templates**: Automated PR markdown compilation complete with Summary, Testing Notes, and Impact matrices.
- **Direct Clipboard Utilities**: Introduced hover-action copy buttons verifying system clipboards without blocking browser popups.
- **Developer Preset Library**: Packaged prefilled change logs to allow rapid performance evaluations on boot.
- **Open-source standard support**: Generated `README.md`, `LICENSE`, `CONTRIBUTING.md`, `CODE_OF_CONDUCT.md`, `SECURITY.md`, and `CHANGELOG.md` templates.
